# 60s-view

> 每天 60 秒看世界，你是懂世界的。

在线地址：https://60s-view.netlify.app

（使用真实接口）

![demo](/.github/img/demo.jpg)

## 功能

- 从 “每天60s读懂世界” 接口获取最新数据
- 图片可下载保存（由于还是 canvas，手机端无法保存图片，working...）

## 技术栈

- React (generated from [Reactease](https://reactease.netlify.app/))

## 数据来源
